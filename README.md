发票开具和查验
=======

### 多渠道发票开具流程:

---

因为大部分的用户都只开通了一种开票渠道，故渠道在提交过来时就已确定

渠道大象，航信，百望 ，支持同步异步两种类型开票

支持主动回调，主动查询

支持票开具成功后择优或者指定邮件和短信渠道进行通知

###多渠道OCR和查验流程：

---

根据成功率和响应时长选择优先队列(遍历所有OCR和查验第三方直至获取结果或者流程退出)

主要渠道有 百度OCR 阿里OCR

OCR有精确票据识别和通用财会识别

票通查验，信天游查验机票...
